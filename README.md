# آمار سبک فارسی (my-fa-stats)

افزونه‌ای سبک برای وردپرس جهت رهگیری نشست‌ها (Session Analytics) با تمرکز روی منبع ورودی، مدت حضور و صفحه خروج.

## نصب
1. پوشه افزونه را با نام `my-fa-stats` داخل مسیر `wp-content/plugins/` قرار دهید.
2. در پیشخوان وردپرس به **افزونه‌ها** بروید و افزونه **آمار سبک فارسی** را فعال کنید.
3. از منوی **آمار سبک** گزارش ماهانه، تنظیمات و صفحه راهنما را مشاهده کنید.

## فراخوانی برنامه‌نویسی‌شده ثبت تغییرات
```php
my_fa_stats_add_changelog( '1.2.1', 'متن تغییرات به فارسی...' );
```

## نکات UTM و Referrer
- برای انتساب دقیق منبع، در لینک کمپین‌ها از `utm_source`، `utm_medium` و `utm_campaign` استفاده کنید.
- در برخی اپ‌ها یا مرورگرها ممکن است `referrer` ارسال نشود؛ در این حالت نشست می‌تواند به‌صورت `direct/none` ثبت شود.
